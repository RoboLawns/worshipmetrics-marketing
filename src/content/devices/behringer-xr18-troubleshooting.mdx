# Behringer XR18 Not Connecting: App and Network Troubleshooting

Since the XR18 is controlled entirely through the X Air app over a network connection, if the app can't find the mixer or the connection drops, you can't mix. Network issues are the most common problem churches encounter with the XR18. Fortunately, most problems follow predictable patterns with straightforward solutions.

## The XR18 Isn't Appearing in the Device List

When you launch X Air and it doesn't show your XR18 in the device list, the mixer either isn't connected to the network or something is blocking the connection between the tablet and mixer.

First, verify the XR18 is powered on. This sounds obvious, but double-check. Look for indicator lights on the front panel. If there's no light at all, plug in the power cable and flip the power switch.

Second, verify your tablet is connected to a network. Check that you're on Wi-Fi or Ethernet. If on Wi-Fi, verify the network name matches the XR18's network. Both devices must be on the same network for the app to find the mixer.

Third, restart the X Air app. Sometimes the app gets stuck in its search process. Close it completely (double-click the home button on iPad and swipe up; on Android, use the recent apps button). Relaunch the app.

Fourth, check that the XR18 is actually connected to the network. On the XR18's front panel, look for network indicator lights. If using Wi-Fi, you should see a signal icon. If using Ethernet, check that the cable is firmly seated in the rear panel port. Wiggle the cable slightly to ensure good contact.

Fifth, if the cable is connected but no indicator light shows, the network port might be malfunctioning. Try a different Ethernet cable. If that doesn't work, contact Behringer support about a possible hardware issue.

## Network Configuration on the XR18

To verify the XR18's network settings, access the mixer's configuration menu directly. Look for a Setup button on the XR18's front panel (it's usually small and recessed). Press it and navigate to Network settings using the limited controls available. You should be able to see whether the mixer has an IP address.

A healthy IP address looks like "192.168.1.50" or similar. If the IP shows "0.0.0.0" or "169.x.x.x," the mixer isn't getting an address from your DHCP server. This means either the network connection isn't working, or your DHCP server isn't configured correctly.

Restart your network router. Power it off completely, wait 30 seconds, and power it back on. This often fixes DHCP issues. Wait for the router to fully boot (usually 1-2 minutes), then check the XR18's network status again.

If the XR18 still shows no valid IP address, try a manual IP assignment. Access the XR18's network settings (through the app or the Setup button), and manually set an IP address like "192.168.1.100." This bypasses the DHCP server and gives the mixer a fixed address. The app should now be able to find the mixer.

## App Still Can't Find the Mixer

If you've verified the XR18 is powered on and connected to the network, and the app still doesn't find it, try manually adding the mixer. In the X Air app, look for a settings or device configuration option. You should see a "Manually Add Device" or similar option.

Click it and enter the XR18's IP address. To find the IP address, you can access the XR18's Setup menu and check the Network section. Write down the IP address you see there (something like "192.168.1.50"), then enter it into the app.

The app should now connect to the mixer directly using the IP address rather than relying on automatic discovery.

## Connection Drops During Operation

If the app connects initially but then loses connection during mixing, you have a network stability problem. This usually happens with Wi-Fi connections. The signal is marginal, and any interference causes dropouts.

Immediately move closer to the Wi-Fi access point. Even a few feet of distance can make a difference. If you're mixing from across the sanctuary, this might not be practical, which indicates you need to upgrade your network.

Switch to Ethernet if possible. If the XR18 is currently on Wi-Fi, run an Ethernet cable from your router to the mixer. This eliminates the Wi-Fi variable and usually resolves dropout issues immediately.

If you must use Wi-Fi, verify there's no interference. Microwave ovens, cordless phones, and other wireless devices can degrade Wi-Fi. Keep these away from your audio control area.

Reduce the number of other devices on your Wi-Fi. If your guest Wi-Fi, church office computers, and security cameras are all sharing the same access point, bandwidth contention might cause dropouts. Separate your audio network from other traffic if possible.

## Latency and Slow Response

If the app connects but controls seem slow to respond—you tap a fader and it takes a second to move, or parameters take time to update—you have network latency.

On Ethernet, this is unusual. Try a different Ethernet cable or verify the cable is firmly seated.

On Wi-Fi, latency usually means weak signal or interference. Move closer to the access point. Look for a clearer Wi-Fi channel if you have access point settings.

Reduce background network traffic. Stop large downloads, streaming video, or backups that might be consuming bandwidth.

## Multiple Tablets Can't Connect Simultaneously

Rarely, when multiple tablets try to connect to the XR18 simultaneously, the app fails or only one tablet succeeds. This is usually a network bandwidth or access point capacity issue.

Verify your access point can handle multiple client devices. Most modern routers support 20-30+ simultaneous connections without issue. If you have an older access point, upgrade to a newer model.

Check whether you've accidentally set a connection limit in the XR18's settings. Most modern network settings don't impose client limits, but check to be sure.

If one tablet connects but another doesn't, restart the tablet that won't connect. Quit the app and relaunch it.

## USB Audio Stops Working After Connection Issues

If you've had Wi-Fi or network issues, and now USB audio to your streaming computer isn't working, there might be a hardware conflict or the USB protocol got interrupted.

Unplug the USB cable from the XR18. Wait 10 seconds. Plug it back in. This resets the USB connection.

On your computer, unplug the mixer from USB, open Device Manager or System Preferences, and remove the Behringer XR18 audio device from the list. Then plug the USB back in. Your computer will redetect it as a fresh device.

Verify that USB Audio is enabled on the XR18. Go to the network or IO settings and confirm USB Audio is toggled on.

## The App Crashes When Trying to Connect

If X Air crashes the moment you tap on the XR18 device name, the app might be corrupted or there's an incompatibility.

Update the X Air app to the latest version. Open the App Store (iOS) or Google Play (Android), search for X Air, and check whether an update is available. Install the latest version.

If updating doesn't fix it, uninstall and reinstall the app. Delete the app from your device completely, restart the device, then download and install the app fresh from the App Store or Google Play.

If crashes persist, the app might be incompatible with your tablet's OS version. Ensure your iPad or Android tablet is running a recent OS version. Very old tablets might not support the latest X Air app.

## No Response From the XR18 After Long Service

If the XR18 has been mixing successfully for an hour or more but suddenly stops responding, the mixer might have overheated or run into a resource limit.

Power cycle the XR18 completely. Turn off the power switch, wait 30 seconds, and turn it back on. This resets everything and usually resolves temporary issues.

If the mixer is in an enclosed space (like a cabinet), verify it has adequate ventilation. The XR18 generates a small amount of heat. If it's completely enclosed, it can overheat. Ensure there's airflow around the mixer.

If overheating is recurring, consider mounting the mixer in a location with better air circulation, or leaving its immediate area open to prevent heat buildup.

## Preventing Network Problems

Once you have everything working, maintain reliability by following these practices:

1. Use Ethernet for the mixer whenever possible. It's the most stable connection type.
2. Keep your Wi-Fi access point firmware updated.
3. Avoid placing the mixer in enclosed spaces without ventilation.
4. Keep network passwords strong and change them periodically.
5. Back up your XR18 settings regularly (through the app's backup function or manual export).
6. Have a second tablet with X Air installed as a backup control device.
7. Test your network setup thoroughly before relying on it for a live service.

## When to Call for Help

If you've worked through the troubleshooting above and the problem persists, it's time to contact Behringer support. They can provide firmware updates, diagnose hardware issues, or provide more detailed technical assistance.

Most XR18 network issues resolve with the troubleshooting steps here. The XR18 is fundamentally reliable. Network problems are usually solvable. Take time to diagnose systematically, and you'll restore functionality quickly.
