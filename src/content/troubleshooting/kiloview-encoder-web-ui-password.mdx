You can't access your Kiloview encoder's web interface. Either the browser won't load the page, or you're locked out because you don't remember the password. You need to configure streaming settings and you're stuck at the login screen. Let's get you back in.

<ProductCTA variant="invoke" />

## Step 1: Find the Kiloview's IP Address

First, discover what IP your Kiloview has:

**Using Kiloview Discovery Tool:**
1. Download **Kiloview Discovery Tool** from Kiloview's website
2. Run it on a PC connected to the same network as the Kiloview
3. The tool will scan and list all Kiloview devices
4. Look for your encoder and note its **IP address**

**Using Router DHCP Table:**
1. Log into your router (usually 192.168.1.1)
2. Find **Connected Devices** or **DHCP Clients**
3. Look for a device with MAC address matching the Kiloview (printed on the back)
4. Note the **IP address** assigned

**Using IP Scanner:**
1. Download **Angry IP Scanner** or **Nmap**
2. Scan your network range (e.g., 192.168.1.0 to 192.168.1.254)
3. Look for "Kiloview" in the hostname or IP list
4. Note the IP

Once you have the IP, move to step 2.

## Step 2: Try Accessing the Web UI

1. Open a web browser (Chrome, Edge, Firefox — avoid Internet Explorer)
2. In the address bar, type `http://[IP_ADDRESS]` (e.g., `http://192.168.1.50`)
3. Press Enter
4. You should see the Kiloview login page

If the page loads, you have a password issue (skip to step 3). If the page doesn't load, continue to step 2B.

**Step 2B: If the page won't load:**
1. Try a different browser (sometimes one works better than others)
2. Clear browser cache: press **Ctrl+Shift+Delete** (Windows) or **Cmd+Shift+Delete** (Mac)
3. Try a different PC on the same network (verifies it's not your PC's issue)
4. Verify the IP address is correct using ping:
   - Open Command Prompt and type `ping [IP]`
   - If you get "Request timed out," the Kiloview is not responding
   - If you get responses, the device is online — try again with the browser

## Step 3: Log In

If the login page loads:

1. Enter the default username: **admin**
2. Enter the default password: **admin** (or try blank password)
3. Click **Login**

If the defaults don't work, you may need to reset the password (step 4).

## Step 4: Reset the Password

If the default credentials don't work, factory reset the Kiloview:

1. **Locate the reset button** on the Kiloview (usually a small recessed button on the back)
2. **Power on the device**
3. **Hold the reset button** for 10–15 seconds while powered on
4. **Release the button** and wait for the device to restart
5. The device will now have factory default credentials (username: **admin**, password: **admin**)
6. Try logging in again

**Warning:** Factory reset erases all streaming configuration. You'll need to reconfigure the Kiloview afterward.

## Step 5: If Password Reset Doesn't Work

Some Kiloview models require a more specific reset procedure:

1. Check the **Kiloview manual** for your specific model (E1 vs N2 have different reset procedures)
2. There may be a **console port** (serial or USB) for resetting via command line
3. Or contact **Kiloview support** with your device MAC address — they can help with password recovery

## Step 6: Verify Network Connectivity

If reset doesn't work and you still can't access:

1. Power off the Kiloview
2. Restart your network switch
3. Power the Kiloview back on
4. Wait 60 seconds for full boot
5. Try accessing the web UI again

A clean power cycle can resolve stuck connection states.

## Step 7: Test with Different Browser and Device

Network or browser issues can block access:

1. Try accessing from a **different PC** on your network
2. Try a **different web browser** (Chrome, Edge, Firefox)
3. If it works on a different device, the issue is specific to your PC or browser
4. If it works on a different browser, clear cache in your regular browser and try again

## Step 8: Check for Firewall Blocking

Firewall may be blocking HTTP access:

1. Temporarily disable Windows Defender Firewall
2. Try accessing the web UI
3. If it works, firewall was blocking it — re-enable and add an exception for port 80

## If You're Locked Out of a Configured Device

If the Kiloview is already configured and streaming, and you're locked out:

1. **Don't factory reset** if you want to preserve streaming settings
2. Contact **Kiloview support** — they may be able to recover the password via account verification
3. Or if the device is still streaming, you can configure a different one while Kiloview support helps with the locked device
