## Step 1: Discover the Camera's IP Address

First, find what IP address the camera has on your network:

**Using AVKANS Discovery Tool:**
1. Download **AVKANS Camera Discovery Tool** from AVKANS's website
2. Run it on a PC connected to the same network as the camera
3. The tool will scan and list all AVKANS cameras on the network
4. Look for your camera (it might be labeled by MAC address if not yet named)
5. Write down the **IP address** shown (e.g., 192.168.1.100)

**Using Router's DHCP Table:**
1. Log into your router (usually 192.168.1.1)
2. Find **Connected Devices** or **DHCP Clients**
3. Look for a device with MAC address matching your camera (printed on the back)
4. Note the IP address assigned
5. Write it down

**Using Manual Scan (Angry IP Scanner):**
1. Download and run **Angry IP Scanner**
2. Set the scan range to your network (e.g., 192.168.1.0 to 192.168.1.254)
3. Run the scan
4. Look for the camera by scanning the results (or look for "AVKANS" in the hostname column)
5. Write down the IP

Once you have the IP, move to step 2.

## Step 2: Test Network Connectivity to the Camera

Before trying the web browser, verify the camera is reachable:

1. Open **Command Prompt** (Windows) or **Terminal** (Mac)
2. Type: `ping [CAMERA_IP]` (e.g., `ping 192.168.1.100`)
3. You should see responses like "Reply from 192.168.1.100: bytes=32 time=2ms"
4. If you get "Request timed out" or "Host unreachable," the camera is offline or unreachable
5. If you get responses, the camera is online — move to step 3

If the ping fails, the camera is either:
- Powered off
- Not on the network
- On a different subnet than your PC
- IP address is wrong

## Step 3: Access the Web Interface

1. Open a **web browser** (Chrome, Edge, Firefox — avoid Internet Explorer)
2. In the address bar, type: `http://[CAMERA_IP]` (e.g., `http://192.168.1.100`)
3. Press Enter
4. You should see the AVKANS camera login page

If the page doesn't load, continue troubleshooting below.

## If the Page Won't Load

**Check browser compatibility:**
- Try a different browser (Chrome usually works best)
- Avoid Internet Explorer (very old, may not load)
- Edge and Firefox should work

**Clear browser cache:**
1. In your browser, press **Ctrl+Shift+Delete** (Windows) or **Cmd+Shift+Delete** (Mac)
2. Click **Clear all**
3. Try accessing the web UI again

**Check firewall:**
1. Is a firewall blocking port 80 (standard web port)?
2. Temporarily disable Windows Defender Firewall and try again
3. If the UI loads, whitelist the camera's IP in your firewall

**Try a different PC:**
1. If you have a second computer, try accessing the camera from that PC
2. Does the web UI load on the other PC?
   - **Yes:** Your streaming PC has a firewall or connection issue
   - **No:** The camera's web server may not be responding

## Step 4: Verify Same Subnet

The PC and camera must be on the same subnet to access the web UI:

1. On your PC, open Command Prompt and type `ipconfig`
2. Find your network adapter's **IPv4 Address** (e.g., 192.168.1.50)
3. Compare to the camera's IP (e.g., 192.168.1.100)
4. The **first three octets** should match (192.168.1.*)
5. If they don't match (e.g., 192.168.0.x vs 192.168.1.x), they're on different subnets
6. Solution: Connect your PC to the same network segment as the camera, or configure the camera's IP to match your subnet

## Step 5: Try the Factory Default IP

If the camera seems to have no IP or is unreachable, it may still be on factory defaults:

1. Most AVKANS cameras ship with a default IP like **192.168.0.110** or **192.168.0.111** (check manual)
2. Try accessing `http://192.168.0.110` in your browser
3. If it loads, you've found it — now reconfigure the IP to your network's range
4. If it doesn't load, the camera may have already been configured

## Step 6: Perform a Factory Reset

If nothing else works, factory reset the camera:

1. Power on the camera
2. Locate the **Reset button** on the back (usually small, recessed, requires a paperclip)
3. Press and hold the reset button for 10–15 seconds while the camera is powered on
4. Release the button and wait for the camera to restart
5. The camera will now be on factory default IP
6. Try accessing `http://192.168.0.110` or `http://192.168.0.111` (check manual for exact default IP)
7. Once you access it, configure the IP to your network's range

**Warning:** Factory reset erases all camera settings (presets, protocols, credentials). Only do this as a last resort.

## Step 7: Update Camera Firmware

If the web UI loads but is acting strange or slow:

1. Log into the camera's web interface (default username/password: often `admin` / `admin`)
2. Look for **Settings > Maintenance > Firmware Update**
3. If a newer firmware is available, download it from AVKANS's support page
4. Upload the firmware file through the web UI
5. Wait for the update to complete (usually 2–5 minutes)
6. The camera will restart
7. Try accessing the web UI again

## Step 8: Check Default Credentials

If the web UI loads but asks for a password:

1. Default username: **admin**
2. Default password: **admin** (or blank password — try pressing Enter)
3. If these don't work, check your AVKANS manual or contact support
4. If the camera has been in use, someone may have changed the password

## Troubleshoot Specific Error Messages

**"Connection Refused":**
- The camera is reachable but the web server isn't responding
- Try a factory reset

**"Request Timed Out":**
- The camera isn't responding to pings
- It's either powered off or not on the network

**Page loads but shows error/blank:**
- Firmware may be corrupted
- Try a factory reset or firmware update

**Asks for password but wrong credentials:**
- Password was changed by someone else
- Contact your IT or whoever set up the camera
